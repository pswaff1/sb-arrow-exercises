Springboard Arrow Functions Exercises

Exercise 1

Instructions

Refactor the following ES5 Map Callback function to instead use two arrow functions:

function double(arr) {
 return arr.map(function (val) {
  return val * 2;  });
}

Solution

const double = (arr) => arr.map((val) => val * 2);

Exercise 2

Instructions

Refactor the following ES5 function to use only arrow functions:

function squareAndFindEvens(numbers) {
 var squares = numbers.map(function(num) {
  return num ** 2;
 });
 var evens = squares.filter(function(square) {
  return square % 2 === 0;
 });
 return evens;
}

Solution

const squareAndFindEvens = (numbers) => numbers.map((num) => num ** 2).filter(square => square % 2 === 0);
